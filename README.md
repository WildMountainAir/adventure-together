## About The Project

In a relationship it is easy to lose track of all of the memories you make with eachother. Adventure Together was created to make memories and reflect on past excursions.

### Planned Future Features:
* Add new adventures to your wishlist
* Track completed adventures
* Upload adventure photos
* Write a date summary and share your favorite memory
* Reflect on past memories

This project is in it's infancy and new features will be added. You may suggest changes by forking this repo and creating a pull request or by opening an issue.

A list of commonly used resources used while working on the Adventure Together project are listed in the acknowledgements.
